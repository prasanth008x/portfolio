Humanoid Robot

Overview:

A semi-autonomous humanoid robot capable of gesture and voice recognition. It can perform obstacle detection, simple movement commands and user-defined tasks.

Technologies Used:

  • Arduino Uno
  • Servo Motors (For Joint Movement)
  • Ultrasonic Sensor (Distance Measurement)
  • Bluetooth Module (HC-05)
  • Voice Recognition Module
  • IR Sensor
  • Battery Pack, Wires, and Robot Frame

Capabilities:

  • Voice-based task execution
  • Obstacle avoidance using ultrasonic sensor
  • Gesture-controlled movement
